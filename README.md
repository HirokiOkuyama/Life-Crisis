# Life-Crisis（製作中）
(2018/07/06 完成予定)
# ゲーム説明
2Dアクションゲームです。  
キャラクターをゴール地点まで誘導すればクリアとなります  
ステージは全部で3つあります。  
上下キーを押すと道が回転し、行き止まりだった場所が通れるようになることがあります。
美味く道を切り替えてゴールを目指しましょう！  

# ・操作説明  
操作はキーボードのみです。  
　左右キー・・・キャラクターの進行方向移動  
　上下キー・・・キャラクターの奥行方向移動  
　Zキー・・・決定  
　Xキー・・・キャンセル  
　Spaceキー・・・ジャンプ
